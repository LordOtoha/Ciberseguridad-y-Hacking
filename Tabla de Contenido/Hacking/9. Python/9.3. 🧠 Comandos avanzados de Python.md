## 游댳 Zipear listas

La funci칩n **`zip()`** se usa para combinar varias listas (o iterables) en una sola. **`zip()`** empareja los elementos de las listas en tuplas, donde el primer elemento de la primera lista se empareja con el primer elemento de la segunda lista, el segundo elemento de cada lista se empareja entre s칤, y as칤 sucesivamente.

```python

#En el ciclo **`for`**, cada vez se desempaquetan las tuplas generadas por **`zip()`**, y asigna sus valores a las variables `num` y `palabra`, respectivamente.
```

```python

3 tres
```

Este m칠todo es muy 칰til cuando tienes dos o m치s listas que contienen datos relacionados, y quieres combinarlas para iterar sobre ellas al mismo tiempo.
### **Resumen:**

- **`enumerate()`**: Se usa cuando necesitas tanto el 칤ndice como el valor de los elementos mientras iteras sobre una lista.
